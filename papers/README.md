# 📄 论文阅读

学习过程中收集的论文 / 资料,统一放在这里。每篇文件以**自包含的中文笔记**形式保存,
包含论文信息、原文链接、章节摘要、个人理解,方便日后回查。

> **添加新论文的约定**
> 1. 文件命名:`<第一作者或简称>_<关键词>_<语言后缀>.md`
>    - 例:`Attention_Is_All_You_Need_CN.md`、`ResNet_CN.md`
> 2. 顶部写明:**标题 / 作者 / 会议 / arXiv / 原文链接**
> 3. 在本 README 表格里登记一行,标注**关联章节**(方便读者跳读)
